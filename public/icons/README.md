# PWA Icons

To generate PWA icons, you can use:
1. https://realfavicongenerator.net/
2. Upload your logo (at least 512x512px)
3. Generate icons for all sizes
4. Download and place them in this directory

Required sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

For now, you can use placeholder icons or generate them from your club logo.
